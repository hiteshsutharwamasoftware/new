## Description

Homepage visitors (both new and returning users) want the ability to switch between dark and light themes to suit their visual preferences and improve accessibility. Providing a theme toggle enhances user experience, accommodates different lighting environments, and aligns with modern UI expectations. This feature should be easily accessible on the homepage and remember the user's selection for future visits.

## Acceptance Criteria

- [ ] Given I am on the homepage, When I view the interface, Then I see a toggle button for switching between dark and light themes in a prominent location.
- [ ] Given the homepage is displayed in light mode, When I click the toggle button, Then the theme changes to dark mode and the UI updates accordingly.
- [ ] Given the homepage is displayed in dark mode, When I click the toggle button, Then the theme changes to light mode and the UI updates accordingly.
- [ ] Given I have selected a theme, When I revisit the homepage or refresh the page, Then my previously selected theme is retained.
